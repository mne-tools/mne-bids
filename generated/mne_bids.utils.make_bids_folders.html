    mne_bids.utils.make_bids_folders¶

    mne_bids.utils.make_bids_folders(subject, session=None, kind=None, root=None, make_dir=True, overwrite=True, verbose=False)[source]¶

    Create a BIDS folder hierarchy.

    This creates a hierarchy of folders within a BIDS dataset. You should plan to create these folders inside the root folder of the dataset.

    Parameters:
    subject : str

    The subject ID. Corresponds to “sub”.

    kind : str

    The kind of folder being created at the end of the hierarchy. E.g., “anat”, “func”, etc.

    session : str | None

    The session for a item. Corresponds to “ses”.

    root : str | None

    The root for the folders to be created. If None, folders will be created in the current working directory.

    make_dir : bool

    Whether to actually create the folders specified. If False, only a path will be generated but no folders will be created.

    overwrite : bool

    If make_dir is True and one or all folders already exist, this will overwrite them with empty folders.

    verbose : bool

    If verbose is True, print status updates as folders are created.
    Returns:
    path : str

    The (relative) path to the folder that was created.

    Examples

    >>> print(make_bids_folders('sub_01', session='my_session',
                            kind='meg', root='path/to/project', make_dir=False))  # noqa
path/to/project/sub-sub_01/ses-my_session/meg
